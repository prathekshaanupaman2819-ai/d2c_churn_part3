# d2c_churn_part3
Part 3: Predictive Churn Analysis & Customer Profiling
​This repository contains the predictive churn analysis and customer profiling models for the D2C project, building upon the foundations laid in "50f0ac8c-912c-4589-ba61-ecda63ae6f38".
​Project Contents
​churn_prediction.ipynb: The Jupyter notebook containing the full predictive logic, churn driver identification, and customer profiling synthesis.
​churn_master_data.csv: The final unified dataset containing integrated RFM, support, campaign, web activity, and return rate metrics.
​How to Run
​Clone this repository to your local machine or environment.
​Install the required libraries using pip install -r requirements.txt.
​Open and run the churn_prediction.ipynb file.
​Academic Integrity
​Analysis is grounded in the provided datasets and supported by visual evidence.
